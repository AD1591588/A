# تطبيق إدارة البضائع (Flutter)

تطبيق أندرويد لإدارة البضائع المنصرفة والمتبقية مع تسجيل اسم المستلم والبائع والسعر.

## المتطلبات
- Flutter 3.0 أو أحدث
- Android Studio أو VS Code

## التشغيل
```bash
flutter pub get
flutter run
```

## المميزات
- إضافة بضاعة جديدة
- عرض وتعديل البضائع
- واجهة عربية بالكامل
- تخزين البيانات محليًا (Hive)
